@media queries

CSS uses the @ media rule tp detect information about the device being used to acces the website. For example, it can be used to detect the size of the device, its capabilities, and its orientation. For this course, we will be using it to detect the device size, but keep in mind that it is capable of much more.

Desktop: Width greater than or equal to 1024px

Tablet: width between 481px and 1023px

Phone: Width is less than or equal 480
